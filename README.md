# 🔐 Login & Signup Page UI

A modern and responsive Login & Signup page built using HTML and CSS with a glassmorphism design effect.

## ✨ Features

- Beautiful glassmorphism UI
- Responsive design
- Background image with blur effect
- Icon integration using Boxicons
- Login & Signup pages connected
- Clean and minimal code structure

## 🛠️ Technologies Used

- HTML5
- CSS3
- Boxicons
